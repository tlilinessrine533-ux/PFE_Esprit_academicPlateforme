# Guide Frontend Angular - Etape 1

Voici exactement ce que vous devez faire pour lancer l'interface Angular.

## 1. Verifier le backend

Avant tout, assurez-vous que le backend Spring Boot tourne sur :

```text
http://localhost:8081
```

Si besoin, relancez-le dans le dossier `backend` avec :

```powershell
& "C:\Maven\apache-maven-3.9.14\bin\mvn.cmd" spring-boot:run
```

## 2. Ouvrir un terminal frontend

Ouvrez un nouveau terminal PowerShell dans :

```text
C:\Users\Asus\Desktop\NV PFE\frontend
```

Puis tapez :

```powershell
cd "C:\Users\Asus\Desktop\NV PFE\frontend"
```

## 3. Lancer Angular

Tapez exactement :

```powershell
npm.cmd start
```

Attendez jusqu'a voir un message indiquant que l'application est servie localement.

## 4. Ouvrir l'application

Dans votre navigateur, ouvrez :

```text
http://localhost:4200
```
